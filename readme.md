# Lunar Lich

Lunar Lich is a game created using [Love2D](https://love2d.org/) for the [Gameboy Jam 5](https://itch.io/jam/gbjam-5).
The game is heavily inspired by classic NES games such as Gun Smoke and Super Contra.

## Controls
WASD - Movement

M - Shoot right

N - Shoot left

M + N - Shoot up
